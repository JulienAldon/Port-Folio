EscapeTheVolcano (jeu vidéo)

Jeu vidéo réalisé avec Unity.
En quatrième année, en Russie j'ai eu l'occasion de suivre des cours de game design. Nous avions à designer un jeu à l'aide de mot aléatoirement choisis. Pour ma part j'ai eu les thèmes suivants :

     - Volcano
     - Cyberpunk
     - Cross-platform
     - RPG

Le but du jeu est de créer son équipe de robots, d'entrer dans le volcan, récuperer le trésor et enfin s'échapper avant que la lave ne nous rattrappe !
Le thème cyberpunk se retrouve dans les graphismes du jeu, les personnages sont des robots.
L'aspect RPG est évoqué par les statistiques des presonnages robots que nous recrutons. Les robots que nous recrutons peuvent être gardés sur plusieurs parties.
Le jeu se veux "simple" dans ses contrôles pour permettre une possibilitée de cross-platform entre PC et mobile. Le jeu est cependant disponible sur linux, mac et windows.

Une fois le jeu designé j'ai décidé de le réaliser en python , puis avec Unity constatant les limitations de pygame.

JAMMEE (site web)

Site Web réalisé en python (backend), React (front end), et javascript (calculs audio).
C'est un projet de fin d’étude "EIP" (Epitech Innovative Project) réalisé sur trois ans (de la 3ème année à la 5ème année) en groupe de 8 personnes.
Le projet vise à reproduire un logiciel de musique assistée par ordinateur dans un navigateur.
Ce projet se destine à un public novice apeuré par la complexitée d'un logiciel moderne.

Publify (site web)

Projet réalisé en Python utilisant l’api spotify pour simplifier la gestion de playlists publiques et collaboratives.
L'application contient deux outils :

   - Le premier outil sychronise une playlist publique et une playlist collaborative, cette action supprime les doublons dans la playlist publique nouvellement synchronisée.
   - Le second outil crée une nouvelle playlist avec tous les nouveaux morceaux des artistes présents dans la playlist.

NanoTekSpice (C++)

Réalisé en C++ en deuxième année à Epitech.
Ce projet est un simulateur de composants électroniques (de circuits simples).
Les portes implémentées sont les suivantes :
      - Input
      - Output
      - Clock
      - True
      - False
      - comp4081 : 2 input AND
      - comp4001 : 2 input NOR
      - comp4011 : 2 input CMOS NAND
      - comp4071 : 2 input OR
      - comp4030 : 2 input EXCLUSIVE-OR
